# Claim Credential

This is used to claim a verifiable credential.

## Import
```jsx
import { ClaimCredential } from "station-ui";
```

## Claim Credential Props

- **`icon`** - <RequiredText /> - `string` - Issuer DAO icon url.
- **`label`** - <RequiredText /> - `string` - A field in the credential.
- **`type`** - <RequiredText /> - `string` - The type of the credential.
- **`issuedBy`** - <RequiredText /> - `string` - The name of the DAO.
- **`expirationDate`** - <RequiredText /> - `string` - When the credential is supposed to expire.
