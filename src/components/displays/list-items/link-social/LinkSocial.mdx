# Link Social

##### Displays a single social component to link to the account.

## Props
- **`icon`** - <RequiredText /> - `ReactNode` - The social icon.
- **`connected`** - <RequiredText /> - `boolean` - Shows a green checkmark if the social is already linked or not.
- **`title`** - <RequiredText /> - `string` - Social name.
- **`username`** - `string` - Shows the user's username.

## Import
```jsx
import { LinkSocial } from "station-ui";
```
