# Credential

##### Displays a single credential item.

## Props
- **`variant`** - Changes the icon on the chain.
  - Options are `claimed`, `unclaimed`.
  - ***You can pass in nothing and there will not be a checkmark on the icon***
- **`chain`** - <RequiredText /> - `**object below**` - Credential icon.
  - `{ icon: string, label: string }`
- **`type`** - <RequiredText /> - `string` - String that is displayed inside of the Pill.
- **`secondaryPill`** - `ReactNode` - A secondary pill in line with the "type" pill. `(Pass in <Pill />)`
- **`description`** - `string` - A string that describes basic info of the credential.
- **`additionalDescription`** - `string` - User generated description for the credential.
- **`title`** - `string` - A string used as the header of the credential.
- **`onRejectClick`** - `() => void` - Function that is called when the reject button is clicked.
- **`onClaimClick`** - `() => void` - Function that is called when the claim button is clicked.


## Import
```jsx
import { Credential } from "station-ui";
```
