# Chain Loader

Displays a loader bar for chains that are loading.

## Import
```jsx
import { ChainLoader } from "terra-money/station-ui"
```

## Chain Loader Props
- **`count`** - <RequiredText /> - `number` - Current chains that have been successfully loaded.
- **`total`** - <RequiredText /> - `number` - Total chains to be loaded.
- **`textLabel`** - `string` - Label for the chain loader.
