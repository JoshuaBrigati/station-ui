# Profile

This is used to show user data and the linked socials.

## Import
```jsx
import { Profile } from "station-ui";
```

## Profile Props

- **`profileImg`** - <RequiredText /> - `string` - User's profile image.
- **`profileLabel`** - <RequiredText /> - `string` - An alternative tag in case the image doesn't load.
- **`profileCoverImg`** - <RequiredText /> - `string` - User's profile cover image.
- **`name`** - <RequiredText /> - `string` - User's name and lastname.
- **`description`** - <RequiredText /> - `string` - User's breif description.
- **`socials`** - <RequiredText /> - `ReactNode` - User's linked or unlinked socials.
- **`onClickView`** - <RequiredText /> - `() => void` - Show's a preview of the profile.
- **`onClickEdit`** - <RequiredText /> - `() => void` - Show's a modal to edit the profile.
