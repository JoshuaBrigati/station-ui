# NftHeaderCard

##### A Card to display the nft header info.

## Props
- **`address`** - <RequiredText /> - `string` - The address of the wallet.
- **`collectionImg`** - <RequiredText /> - `string` - The collection image associated with the nft.
- **`collectionName`** - <RequiredText /> - `string` - The collection name associated with the nft.

## Import
```jsx
import { NftHeaderCard } from "station-ui";
```
