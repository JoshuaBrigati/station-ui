# NftSaleCard

##### A Card to display the nft sale info.

## Props
- **`collectionPrice`** - <RequiredText /> - `number` - The collection floor price.
- **`currentPrice`** - <RequiredText /> - `number` - The nft current price.
- **`currencySymbol`** - `string` - The symbol of the currency ($).
- **`saleEndDate`** - - <RequiredText /> - `Date` - When the sale ends.
- **`tokenImg`** - <RequiredText /> - `string` - The image url of the token.
- **`tokenName`** - <RequiredText /> - `string` - The name the token.
- **`onBuyNowClick`** - <RequiredText /> - `() => void` - OnClick functionality for the buy now button.
- **`onMakeOfferClick`** - <RequiredText /> - `() => void` - OnClick functionality for the make offer button.
- **`onSaleFinished`** - <RequiredText /> - `() => void` - Callback function when the sale ends.

## Import
```jsx
import { NftSaleCard } from "station-ui";
```
