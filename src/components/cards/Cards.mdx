# ProposalCard [<InLineDocsLink />](/docs/components-cards-proposal--docs)

##### This is used to show a wallets individual delegation information.

## Props
- **`proposal`** - <RequiredText /> - `**object below**` - Data for the proposal to display.
  - **`ProposalHeaderProps`** = `{`
    - **`status`** - <RequiredText /> - `string` - Used to display statup of the proposal.
      - options: "Voting" | "Deposit" | "Rejected" | "Passed"
    - **`title`** - <RequiredText /> - `string` - The title of the page (The large text).
    - **`metaText`** - <RequiredText /> - `string` - The description of the page (The small text on top).
    - **`metaImage`** - <RequiredText /> - `string` - The image of the page (The small image on top).
    - **`submittedDate`** - <RequiredText /> - `string` - The date the proposal was submitted.
    - **`endsDate`** - <RequiredText /> - `string` - The date the proposal ends.
      - ***If this date hasn't passed, a countdown will be displayed***
    - **`voteValue`** - `string` - The value of what the USER voted.
      - option: "Yes" | "No" | "Abstain" | "No (Veto)"
  - `}`
- **`progressData`** - <RequiredText /> - `ProgressBarData[]` - Data for the `<ProgressBar />`.
  - **`ProgressBarData[]`** = `{`
      type: "yes" | "abstain" | "no" | "noWithVeto" | "deposit"
      percent: string
      amount?: string | ReactNode
  - `}`
- **`threshold`** - <RequiredText /> - `number` - The threshold for the ProgressBar.
- **`progressLabelOverride`** - `string` - Override for the progress label.
- **`className`** - `string` - ClassName override for the card.

## Import
```jsx
import { ProposalCard } from "terra-money/station-ui";
```
