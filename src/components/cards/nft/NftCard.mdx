# NftCard

##### A Card to display an nft.

## Props
- **`chainImg`** - <RequiredText /> - `string` - The chain image url associated with the nft.
- **`chainName`** - <RequiredText /> - `string` - The chain name associated with the nft.
- **`currencySymbol`** - `string` - The symbol of the currency ($).
- **`nftImage`** - <RequiredText /> - `string` - The nft main image url.
- **`nftImageLabel`** - <RequiredText /> - `string` - The nft main image label.
- **`nftName`** - <RequiredText /> - `string` - The nft name.
- **`tokenImg`** - <RequiredText /> - `string` - The image url of the token.
- **`tokenName`** - <RequiredText /> - `string` - The name the token.
- **`tokenValue`** - <RequiredText /> - `number` - How many tokens the nft is worth.

## Import
```jsx
import { NftCard } from "station-ui";
```
