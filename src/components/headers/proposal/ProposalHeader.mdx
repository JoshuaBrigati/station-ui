# Proposal Header

##### A Header for displaying information about a proposal.

## Props
- **`status`** - <RequiredText /> - `string` - Used to display statup of the proposal.
  - options: "Voting" | "Deposit" | "Rejected" | "Passed"
- **`title`** - <RequiredText /> - `string` - The title of the page (The large text).
- **`metaText`** - <RequiredText /> - `string` - The description of the page (The small text on top).
- **`metaImage`** - <RequiredText /> - `string` - The image of the page (The small image on top).
- **`submittedDate`** - <RequiredText /> - `string` - The date the proposal was submitted.
- **`endsDate`** - <RequiredText /> - `string` - The date the proposal ends.
  - ***If this date hasn't passed, a countdown will be displayed***
- **`voteValue`** - `string` - The value of what the USER voted.
  - option: "Yes" | "No" | "Abstain" | "No (Veto)"

## Import
```jsx
import { ProposalHeader } from "@terra-money/station-ui";
```
